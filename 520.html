FEHLER 520 - UNBEKANNTER WEB FEHLER

Leider ist bei der Kommunikation mit unseren Web Services ein unerwarteter Fehler aufgetreten.

Dies ist eine catch-all Antwort, dass unsere Web Services etwas unerwartetes zurückgeben.

Dieser Fehler kann aus derartigen Serverausfällen oder Wartungsarbeiten resultieren.

  • Liegt der Fehler an mir?
  • Was kann ich tun? 
  • Dauer

Nein. Fehler vom Codestamm 500 sind serverseitige Probleme. Sie sind nicht der einzigste, der momentan mit unseren Web Services Kommunikationsprobleme erfährt. 

Dieser Fehler kann nur durch Reparaturen an der Webserver-Software behoben werden. Es ist kein Problem auf der Client-Seite. Es ist unsere Aufgabe, diesen Fehler zu beheben.

Grundsätzlich sollten serverseitige Fehler von userem Team recht schnell behoben werden.

Trotz dem kann es immer mal wieder vorkommen, dass die Wartungsarbeiten von derartig schwerwiegenden Fehlern über 24 Stunden andauern.
